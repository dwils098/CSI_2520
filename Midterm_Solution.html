8 SUDOKU

app(X,L) is a predicate that verifies if X is in list L. napp(X,L) is a predicate that verifies if X is NOT in list L.

next(I,J) is a predicate that returns the next element following I and J

nth(I, L, R) is a predicate that returns the ith element in a list L.

column(J, S, X) is a predicate that returns C the column ranked J in the matrix S.

count(N, X, L) is a predicate that counts the elts in a list L.

square( I, J, S, M) is a predicate that returns true if M is the sub-matrix at location (I,J).

place( X, I, J, M, S) is a predicate that placed the elt X at pos J in row I and returns the resulting matrix as S.
