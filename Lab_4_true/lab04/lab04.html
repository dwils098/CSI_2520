Exercise 2. Maximum and Minimum Elements

Write a Prolog program that finds the maximum and minimum element in a list of numbers. The program should work as follows: 

?- maxmin([3,1,5,2,7,3],Max,Min).
Max = 7
Min = 1

?- maxmin([2],Max,Min).
Max = 2
Min = 2

Please hand-in the answer to this question on Virtual Campus by 5:00pm today! Complete the following Prolog program to reverse the order of the elements in the list but also remove every second element. The program should work as follows: 

?- reverseDrop([3,1,5,2,7,3],L).
L = [ 7, 5, 3 ].

?- reverseDrop(['world','a','hello'],L).
L = [ 'hello', 'world' ].
Starter code: 


reverseDrop(L,R) :- reverseDrop(L,1,[],R), !.

reverseDrop([HL|TL],N,I,R) :- 
          N = 1,
          reverseDrop(TL,0,[HL|I],R).
